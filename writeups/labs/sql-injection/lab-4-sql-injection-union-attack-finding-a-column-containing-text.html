🔍 Description

Building on Lab 3 where the number of columns (3) was determined, this lab requires identifying which of those columns accepts string data. A UNION attack can only retrieve text data through columns that are compatible with string types. The technique is to replace each NULL placeholder one at a time with a known string value. When the application returns a valid response containing that string, the compatible column is identified.

The lab provides a specific string (3saQPt) to retrieve, which must appear in the application's response to confirm the correct column.

🎯 End Goals

  • Identify which of the 3 columns returned by the query accepts string data
  • Inject the target string '3saQPt' into the correct column position
  • Confirm the string appears in the response and solve the lab

🪜 Steps to Reproduce

  1. Navigate to the web shop and capture a GET /filter?category=Corporate+gifts request in Burp Suite Repeater.
  2. From Lab 3, we already know the query returns 3 columns.
  3. Test each column for string compatibility by replacing one NULL at a time with the target string '3saQPt':
  4. Try ' UNION SELECT '3saQPt',NULL,NULL-- — if error, column 1 is not a string type.
  5. Try ' UNION SELECT NULL,'3saQPt',NULL-- — application returns valid response with the string visible, confirming column 2 accepts string data.
  6. Lab solved.

🧠 Analysis

Step 1 — Inject string into column 2 — valid response

Sent ' UNION SELECT NULL,'3saQPt',NULL-- in Burp Suite Repeater. The application returned a valid 200 response containing the string 3saQPt in the product listing, confirming column 2 accepts string data.

Step 2 — Lab solved

Browser displayed the lab solved confirmation banner.

🎯 Impact

Identifying a string-compatible column is a critical step in UNION-based SQL injection. Once a text-returning column is confirmed, an attacker can extract arbitrary string data from the database — including usernames, passwords, email addresses, and other sensitive table contents — by substituting database queries in place of the string literal.
